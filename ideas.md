## Ideia 1: "Cyberpunk Terminal"

**Movimento de Design**: Cyberpunk / Hacker Terminal — inspirado em interfaces de terminais de hacking, com estética neon sobre fundo escuro.

**Princípios Centrais**:
1. Fundo escuro profundo com acentos neon (verde/ciano) que remetem a terminais
2. Tipografia monospace para dados técnicos, sans-serif moderna para títulos
3. Elementos com bordas finas brilhantes (glow effect) e animações de "digitação"
4. Layout funcional tipo dashboard de controle de missão

**Filosofia de Cores**: Fundo preto/cinza muito escuro (#0a0a0f) com acentos em verde neon (#00ff88) e ciano (#00d4ff). O contraste extremo comunica "poder técnico" e "automação avançada".

**Paradigma de Layout**: Layout vertical em coluna única com seções empilhadas tipo terminal, com um painel de log ao vivo na parte inferior.

**Elementos Assinatura**: 
- Borda com glow neon pulsante nos cards ativos
- Console de log com scroll automático mostrando cada requisição

**Filosofia de Interação**: Feedback instantâneo tipo terminal — cada ação gera uma "linha de log" visível.

**Animação**: Efeito de digitação nos textos, pulse nas bordas ativas, fade-in sequencial nos logs.

**Sistema Tipográfico**: JetBrains Mono para dados/logs, Space Grotesk para títulos e labels.

## Ideia 2: "Golden Bird Dashboard"

**Movimento de Design**: Premium Dark UI — inspirado no branding do próprio VoaPix (pássaro dourado, fundo roxo/índigo), com acabamento premium e elegante.

**Princípios Centrais**:
1. Paleta fiel ao app original: fundo índigo escuro (#1a1145) com acentos dourados (#f5c542)
2. Cards com glassmorphism sutil e bordas douradas
3. Hierarquia visual clara com o pássaro/moeda como elemento central
4. Dashboard compacto com todos os controles visíveis sem scroll

**Filosofia de Cores**: Índigo profundo como base (#1a1145, #0f0b2e), dourado (#f5c542, #d4a017) como cor de ação e destaque, branco suave para texto. Transmite "riqueza" e "ganhos" — alinhado com o tema de pontos/dinheiro.

**Paradigma de Layout**: Grid assimétrico com painel de configuração à esquerda e painel de status/logs à direita. Header com branding forte.

**Elementos Assinatura**:
- Ícone do pássaro dourado animado no header
- Contador de saldo animado com efeito de "slot machine"
- Indicador de status com pulso dourado quando ativo

**Filosofia de Interação**: Feedback visual rico — botões com hover dourado, sliders com thumb personalizado, toasts elegantes para cada resposta da API.

**Animação**: Counter animado para saldo, pulse no indicador de status, slide-in para logs, hover scale nos botões.

**Sistema Tipográfico**: Outfit para títulos (bold, impactante), Inter para corpo/dados (legível, técnico).

## Ideia 3: "Brutalist Control Panel"

**Movimento de Design**: Neo-Brutalism — formas geométricas ousadas, bordas grossas, cores saturadas, sem pretensão de "suavidade".

**Princípios Centrais**:
1. Bordas grossas pretas (3-4px), sombras duras sem blur
2. Cores saturadas e contrastantes (amarelo vivo, preto, branco)
3. Tipografia pesada e oversized para títulos
4. Layout em blocos empilhados com espaçamento generoso

**Filosofia de Cores**: Fundo branco/creme (#fefce8) com blocos de cor saturada — amarelo (#facc15), preto (#000), vermelho (#ef4444) para alertas. Comunicação direta e sem frescura.

**Paradigma de Layout**: Blocos empilhados verticalmente com bordas grossas, cada seção é um "card brutalist" independente.

**Elementos Assinatura**:
- Sombras duras offset (4px 4px 0 #000)
- Botões com hover que inverte cores completamente

**Filosofia de Interação**: Respostas táteis — botões que "afundam" no click, transições abruptas e intencionais.

**Animação**: Transições de cor abruptas, shake no erro, bounce nos contadores.

**Sistema Tipográfico**: Space Grotesk Black para títulos, DM Sans para corpo.

</idea>
